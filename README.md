# HEKIT - 一个工具集合
HEKIT是一个用Rust编写的工具集合，提供各种实用功能。

## 🚀 功能特性
### 批量重命名工具
- 支持前缀、后缀、序号、替换多种重命名规则
- 预览模式（先查看修改效果，无实际变更）
- 自动处理文件名冲突
- 支持文本替换（简单替换和正则表达式）
- 支持序号生成（自动补零，固定3位补零）
- 支持扩展名修改
- 备份功能（防止误操作）

### 批量搜索工具
- 按文件名模式搜索（支持通配符 *）
- 支持文件类型过滤
- 支持文件大小范围筛选（可指定B/KB/MB/GB单位）
- 按文件修改时间筛选
- 递归搜索子目录（可关闭递归）
- 大小写敏感/不敏感选项
- 高性能BFS目录遍历（适合大目录/多层级目录）
- 支持中断搜索（按Q键停止，无冗余等待）

### 批量压缩工具
- 支持多种压缩格式：ZIP、TAR.GZ、TAR.BZ2
- 可调节压缩级别（1-9级，1级最快/压缩率最低，9级最慢/压缩率最高，默认6级）
- 自定义输出目录（避免压缩文件与源文件混放）
- 预览压缩效果
- 支持单个文件独立压缩或多文件合并压缩

### 批量转换工具
- 支持主流文件格式转换
  - 图片：JPG、PNG、GIF、BMP、WebP 互转
  - 文档：TXT、MD 互转
- 批量处理多个文件（自动过滤非支持格式）
- 自定义转换参数（图片压缩质量、分辨率调整）
- PDF文件信息提取（基础功能）
- 相同格式文件直接复制

### 批量清理工具
- 清理空文件夹、临时文件、旧日志文件
- 安全删除模式（数据覆盖）
- 自定义清理规则
- 备份功能（防止误操作）
- 支持按文件修改时间筛选待清理文件

## 🛠️ 安装与使用方法
### 从源码编译安装
1.  克隆项目
```bash
git clone https://gitee.com/zhanghed/hekit.git
cd hekit
cargo build --release
```

### 直接下载二进制文件（推荐）
1.  访问 [发布页面](https://gitee.com/zhanghed/hekit/releases)

## 📁 项目结构
```
hekit/
├── src/
│   ├── main.rs          # 程序入口点
│   ├── app.rs           # 主应用程序逻辑
│   ├── lib.rs           # 库文件
│   ├── utils.rs         # 工具函数
│   ├── error.rs         # 错误处理
│   ├── version.rs       # 版本检查功能
│   └── features/        # 功能模块
│       ├── clean/       # 批量清理工具
│       ├── compress/    # 批量压缩工具
│       ├── convert/     # 批量转换工具
│       ├── rename/      # 批量重命名工具
│       ├── search/      # 批量搜索工具
│       ├── common.rs    # 公共工具接口
│       └── mod.rs       # 模块导出
├── Cargo.toml           # 项目配置和依赖管理
└── build.rs             # 构建脚本
```
